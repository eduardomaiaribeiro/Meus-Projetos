## 📂 Estrutura

Cada subpasta representa um projeto Java independente, contendo seu próprio código-fonte, arquivos de configuração e, geralmente, um README.md específico com instruções detalhadas.

---

## 🚀 Como executar um projeto Java

1. Certifique-se de ter o [Java JDK](https://www.oracle.com/java/technologies/downloads/) instalado em seu ambiente.
2. Navegue até a pasta do projeto desejado:
	```bash
	cd Caminho/Para/O/ProjetoJava
	```
3. Compile o projeto (caso necessário):
	```bash
	javac Main.java
	```
4. Execute o projeto:
	```bash
	java Main
	```
