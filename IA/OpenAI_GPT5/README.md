
# Experimentos com OpenAI GPT-5

Esta pasta reúne notebooks e exemplos práticos para explorar as novidades, mudanças e aplicações do modelo GPT-5 da OpenAI.

## 📚 Conteúdo

Principais arquivos:

- **01-mudancas.ipynb** — Principais mudanças e novidades do GPT-5
- **02-exemplo.ipynb** — Exemplos básicos de uso do modelo
- **03-preamble.ipynb** — Testes com preâmbulos e instruções customizadas
- **04-exemplo_final.ipynb** — Aplicação prática integrando recursos do GPT-5
- **requirements.txt** — Dependências necessárias para execução

Ambiente virtual sugerido em `vev/`.

## 🚀 Como Executar

1. **Pré-requisitos:**
	- Python 3.x instalado
	- (Opcional) Ambiente virtual ativado

2. **Instale as dependências:**
	```bash
	pip install -r requirements.txt
	```

3. **Configure sua chave de API:**
	- Defina a variável de ambiente `OPENAI_API_KEY` ou utilize um arquivo `.env` com sua chave da OpenAI.

4. **Execute o notebook desejado:**
	```bash
	jupyter notebook NOME_DO_ARQUIVO.ipynb
	```

## 🎯 Objetivo

Demonstrar, de forma prática, as capacidades, diferenças e melhores práticas para uso do GPT-5, servindo como referência para estudos e desenvolvimento de novas aplicações.
