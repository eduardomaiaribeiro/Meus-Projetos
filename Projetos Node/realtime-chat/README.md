# Chat em Tempo Real

Aplicação de chat em tempo real usando Node.js, Express e Socket.io.

## Funcionalidades
- Envio e recebimento de mensagens instantâneas
- Suporte a múltiplos usuários em salas de chat

## Como usar
1. Instale as dependências:
	```bash
	npm install
	```
2. Inicie o servidor:
	```bash
	npm start
	```
3. Acesse: http://localhost:3002

## Como testar
Abra dois navegadores diferentes e envie mensagens para ver o chat funcionando em tempo real.
