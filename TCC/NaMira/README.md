# NaMira

Projeto desenvolvido em Flutter para análise e visualização de dados relacionados à dengue.

## Descrição
O NaMira é um aplicativo multiplataforma (Android, iOS, Web e Desktop) que permite a visualização de boletins epidemiológicos, mapas interativos e dicas de prevenção sobre a dengue. O projeto integra dados reais e simulados, fornecendo informações úteis para a população e profissionais de saúde.

## Instalação
1. Certifique-se de ter o [Flutter](https://docs.flutter.dev/get-started/install) instalado em sua máquina.
2. Clone este repositório:
	```bash
	git clone <URL_DO_REPOSITORIO>
	```
3. Acesse a pasta do projeto:
	```bash
	cd NaMira
	```
4. Instale as dependências:
	```bash
	flutter pub get
	```

## Execução
Para rodar o aplicativo em modo de desenvolvimento:
```bash
flutter run
```
Você pode especificar o dispositivo desejado usando o parâmetro `-d`.

## Estrutura de Pastas
- `lib/` — Código fonte principal do aplicativo Flutter
- `assets/` — Arquivos estáticos (imagens, JSONs)
- `Dados/` — Dados utilizados para análise (boletins, PDFs, JSON)
- `android/`, `ios/`, `web/`, `linux/`, `macos/`, `windows/` — Plataformas suportadas
- `test/` — Testes automatizados
