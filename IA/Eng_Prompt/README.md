
# Engenharia de Prompt — Exemplos e Boas Práticas

Esta pasta reúne notebooks práticos sobre técnicas de engenharia de prompt para modelos de linguagem (LLMs), com foco em aplicações, boas práticas e experimentação.

## 📚 Conteúdo dos Notebooks

Cada notebook aborda um conceito ou técnica fundamental:

- **01-roles.ipynb** — Definição de papéis para o modelo (role prompting)
- **02-params.ipynb** — Parâmetros de temperatura, top-p, etc.
- **03-boas_praticas.ipynb** — Boas práticas para criação de prompts
- **04-output_estruturado.ipynb** — Geração de respostas estruturadas
- **05-condicoes.ipynb** — Condições e instruções condicionais
- **06-few_shot.ipynb** — Exemplos few-shot para guiar o modelo
- **07-steps.ipynb** — Prompts em etapas (step-by-step)
- **08-cot.ipynb** — Chain-of-Thought (raciocínio encadeado)
- **09-sumarizacao.ipynb** — Técnicas de sumarização
- **10-texto_complemento.ipynb** — Complementação de textos
- **11-transformacao_texto.ipynb** — Transformação e reescrita de textos
- **12-analise_textual.ipynb** — Análise e classificação de textos

## 🚀 Como Utilizar

1. Certifique-se de ter o Python 3.x e Jupyter Notebook instalados.
2. (Opcional) Crie um ambiente virtual e instale as dependências:
	```bash
	pip install -r requirements.txt
	```
3. Abra o Jupyter Notebook na pasta:
	```bash
	jupyter notebook
	```
4. Explore os notebooks na ordem sugerida ou conforme seu interesse.

## 🎯 Objetivo

O objetivo desta coleção é servir como material de estudo e referência para quem deseja aprender, praticar e dominar técnicas modernas de engenharia de prompt para LLMs.
