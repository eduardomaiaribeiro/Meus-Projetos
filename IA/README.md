
# Projetos de Inteligência Artificial (IA)

Esta pasta reúne projetos, exemplos e experimentos práticos relacionados à área de Inteligência Artificial, com foco em aplicações modernas de LLMs, engenharia de prompt, integração com APIs e análise de dados.

## 📂 Estrutura da Pasta

Os principais diretórios e seus propósitos:

- **Eng_Prompt/** — Notebooks sobre técnicas de engenharia de prompt para LLMs
- **openai/** — Exemplos de uso da API da OpenAI (chatbots, geração de texto, voz, etc.)
- **OpenAI_GPT5/** — Experimentos e novidades com o modelo GPT-5

Cada subpasta possui seu próprio README.md detalhando exemplos, dependências e instruções específicas.

## 🚀 Como Utilizar

1. Certifique-se de ter o Python 3.x instalado.
2. (Opcional) Crie e ative um ambiente virtual.
3. Instale as dependências de cada projeto conforme o respectivo `requirements.txt`.
4. Siga as instruções do README.md de cada subpasta para executar notebooks ou scripts.

## 🎯 Objetivo

Centralizar materiais de estudo, demonstração e prototipagem de soluções em IA, facilitando o aprendizado e a experimentação com as tecnologias mais atuais do campo.
